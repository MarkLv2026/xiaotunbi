# 小豚当家影锋风格 BI 看板

这是按影锋/锋影 BI 类商业驾驶舱思路重做的 A 方案：看板页面自带上传数据源入口，上传最新 Excel 后自动刷新全页。

## 启动

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 每日维护

1. 导出或整理当天最新 Excel 数据源。
2. 打开 BI 看板网页。
3. 在左侧「数据源更新」上传 Excel。
4. 页面自动刷新经营总览、趋势图、同比/环比、TOP 商品和明细表。
5. 需要留档时，下载 CSV。

## 看板结构

- 经营总览：支付金额、支付件数、支付买家、访客、支付转化率、客单价、退款率
- 全域趋势：金额 + 访客 + 销量组合趋势
- 渠道结构：渠道销售占比
- 店铺/品类排行：店铺销售排行、品类销售排行
- 经营矩阵：品类流量-转化矩阵
- 诊断页：月度同比环比、TOP 商品、渠道/店铺矩阵、款式/型号明细

## 数据源要求

建议包含工作表：`天猫数据源`、`京东抖音数据源`。
核心字段：统计日期、渠道、店铺、品类、型号、款式、商品名称、商品ID、商品访客数、商品浏览量、商品加购人数、商品加购件数、支付买家数、支付件数、支付金额、成功退款金额。
